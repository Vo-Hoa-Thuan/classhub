Hướng dẫn kiểm tra chức năng quên mật khẩu đã được tích hợp vào Frontend

📋 Các bước kiểm tra

Bước 1: Khởi động Backend Server

Đảm bảo Backend server đang chạy trên port 8080:

cd Backend
npm start

✅ Backend đã được cấu hình với email service

Bước 2: Khởi động Frontend

Chạy Frontend React app:

cd Frontend
npm start

✅ Frontend đã được tích hợp với các component quên mật khẩu

Bước 3: Test Flow Quên Mật Khẩu

  1. Truy cập Trang Login
  2. Click vào link "Quên mật khẩu?"
  3. Nhập email hợp lệ (ví dụ: test@example.com)
  4. Click "Gửi link đặt lại mật khẩu"
  5. Kiểm tra email để nhận link reset
  6. Click vào link trong email
  7. Nhập mật khẩu mới theo yêu cầu
  8. Xác nhận mật khẩu mới
  9. Click "Đặt lại mật khẩu"

Bước 4: Kiểm tra các tính năng

  • ✅ Validation email format
  • ✅ Loading states
  • ✅ Error handling
  • ✅ Success messages
  • ✅ Password requirements validation
  • ✅ Show/hide password toggle
  • ✅ Token verification
  • ✅ Responsive design

🔗 Các URL quan trọng

  • Login: http://localhost:3000/login
  • Forgot Password: http://localhost:3000/forgot-password
  • Reset Password: http://localhost:3000/reset-password/[token]

🐛 Troubleshooting

Nếu email không được gửi:

  • Kiểm tra App Password trong Gmail
  • Đảm bảo 2FA được bật
  • Kiểm tra firewall/antivirus
  • Xem logs trong console

Nếu token không hợp lệ:

  • Token có thể đã hết hạn (15 phút)
  • Token đã được sử dụng
  • URL không đúng format
